CREST model V1.x, a distributed hydrologic model, is jointly developed at the University of Oklahoma and NASA.

Note

CREST model V1.x is an initial publication, featuring hydrologic processes including canopy interception, Variable Infiltration Curve for soil infiltration, three cascading soil layers, Linear Reservoir Routing for surface flow and subsurface flow
